# Contribuer

Merci de ton intérêt pour ce projet ! Toute aide est la bienvenue.

## Comment contribuer

### Signaler un bug ou proposer une amélioration

1. Ouvrir une [issue](../../issues) sur GitHub
2. Décrire le problème ou l'idée
3. Si possible, proposer une solution

### Proposer du code

1. Forker le repo
2. Créer une branche (`git checkout -b ma-feature`)
3. Faire les modifications
4. Tester localement (`make test`)
5. Committer (`git commit -m "Ajoute ma feature"`)
6. Pusher (`git push origin ma-feature`)
7. Ouvrir une Pull Request

## Guidelines

### Code

- Scripts en **bash**, compatibles POSIX
- Messages et commentaires en **français**
- Tester avec `bash -n script.sh` avant de committer

### Documentation

- Markdown simple
- Schémas ASCII quand utile
- Exemples concrets

## Questions ?

Ouvrir une issue ou contacter les mainteneurs du projet.
